Directs
A method for solving systems of linear equations is straightforward if defined as a finite set of operations and steps to find the solution. The method generates a solution that would be accurate if it weren't for rounding errors.
Iteratives
The iterative methods for the solution of systems of linear equations of the form Ax=b, are very useful for the approximation of the values of the variables, but it must be taken into account that the matrix A is a non-single matrix of order nxn with elements in the main diagonal not null. The support of the arguments of these methods, is the fixed point method, in this sense it must be understood that an iterative method for the solution of systems of equations behaves that from a vector of initial approximations, generates the solution of the problem by a series of iterations.
